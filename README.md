# Price Monitor Bot

Автоматизированный Telegram-бот на языке Python для мониторинга цен в интернет-магазинах (**Steam**, **Ситилинк**) с системой мгновенных уведомлений и визуализацией динамики стоимости.

---

## ✨ Основные возможности

*   **🔍 Автоматический сбор данных:** Бот самостоятельно проверяет актуальную стоимость товаров по расписанию.
*   **📊 Визуализация изменений:** Генерация наглядных графиков истории цен с помощью библиотеки `Matplotlib`.
*   **🔔 Умные уведомления:** Система оповещает пользователя только при значимых изменениях цены (настраиваемый порог `threshold`).
*   **⚙️ Гибкое управление:** Возможность установки целевой цены, переименования отслеживаемых позиций и выбора темы оформления.
*   **🛡️ Защита от блокировок:** Поддержка ротации прокси-серверов для стабильной работы с крупными торговыми площадками.

---

## 🛠 Технологический стек

*   **Язык программирования:** Python 3.12+
*   **Библиотека для бота:** aiogram 3.x (Асинхронный фреймворк)
*   **Парсинг:** BeautifulSoup4, aiohttp
*   **База данных:** SQLite (библиотека aiosqlite)
*   **Графики:** Matplotlib
