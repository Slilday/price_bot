# 🛍️ Price Monitor Bot

Telegram-бот на языке Python для мониторинга цен в интернет-магазинах (**Steam**, **Ситилинк**) с системой мгновенных уведомлений и с автоматическим построением графиков динамики цен.

## ✨ Основные возможности

*   **Автоматический сбор данных:** Бот самостоятельно проверяет актуальную стоимость товаров по заданному расписанию.
*   **Визуализация изменений:** Генерация наглядных графиков истории цен с помощью библиотеки `Matplotlib`.
*   **Умные уведомления:** Система оповещает пользователя только при изменении цены (настраиваемый порог `threshold`).
*   **Гибкое управление:** Возможность установки целевой цены, переименования отслеживаемых позиций и выбора тем оформления.
*   **Защита от блокировок:** Поддержка ротации прокси-серверов для стабильной работы с крупными торговыми площадками.
*   **Логирование:** Бот ведет историю всех важных событий и ошибок в текстовом файле, что позволяет оперативно выявлять и устранять неполадки.

## 🛠 Технологический стек

*   **Язык программирования:** Python 3.12+
*   **Библиотека для бота:** aiogram 3.x (Асинхронный фреймворк)
*   **Парсинг:** BeautifulSoup4, aiohttp
*   **База данных:** SQLite (библиотека aiosqlite)
*   **Графики:** Matplotlib

## 🚀 Инструкция по запуску

### 1. Клонирование репозитория

```bash
git clone https://github.com/Slilday/price_bot.git
cd price_bot
```

### 2. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 3. Настройка конфигурации
Создайте файл .env в корневой папке и добавьте ваш токен бота:
```code
BOT_TOKEN=ваш_токен_от_BotFather
```

### 4. Запуск программы
```code
python main.py
```

### Структура проекта
* bot/ — обработчики команд (handlers), состояния (FSM) и клавиатуры.
* core/ — логика парсинга сайтов и менеджер прокси-серверов.
* database/ — работа с реляционной базой данных SQLite.
* services/ — фоновый модуль мониторинга и отрисовка графиков.
* proxies.txt — список IP-адресов для обхода ограничений.
